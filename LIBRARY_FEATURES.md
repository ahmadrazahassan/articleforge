## ðŸš€ Usage Workflows

### Workflow 1: Export All Articles
1. Click "Select All" button
2. Choose export format (HTML, MD, JSON, or WordPress)
3. Each article downloads as separate file with article name
4. Files ready for import into CMS or publishing platform

### Workflow 2: Find High-Quality Articles
1. Set Min SEO Score filter to 80
2. Sort by SEO Score
3. View only high-quality articles
4. Export or publish selected articles

### Workflow 3: Organize by Category
1. Filter by specific category
2. Update category for multiple articles
3. Reorganize content structure
4. Export by category

### Workflow 4: Duplicate & Modify
1. Find article to duplicate
2. Click "Duplicate" button
3. Article appears in library with "(Copy)" suffix
4. Edit and export modified version

### Workflow 5: Bulk Export for CMS
1. Select articles to export
2. Choose "Export Each as Separate File"
3. Select format (HTML, MD, JSON, WordPress)
4. All files download with proper naming
5. Import into CMS or publishing platform
